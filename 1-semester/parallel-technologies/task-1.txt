Есть два массива: A, B. Необходимо посчитать поэлементную сумму чисел в массиве.
 

Насколько возможно решить ее параллельным способом? Подумайте над следующими аспектами:

выделение памяти;
подсчет непосредственной суммы.
Какой из факторов может нас ограничить в параллельном решении этой задачи?



a = [a1, a2, a3, a4, a5, a6]

b = [b1, b2, b3, b4, b5, b6]

c = a + b = [a1 + b1, a2 + b2, ..., an + bn]

Без распараллеливания надо будет сделать N операций суммы и N операций добавления в массив

С точки зрения выделения памяти при распараллеливании мы не получим преимуществ

С точки зрения подсчета суммы количество операций на равно 2N / M + M
N / M Операций суммы
N / M Операций добавления в массив
M при слиянии массивов
